
# Step Beat Sequencer 
This is my creative coding project. It is a step sequencer built using Godot. It allows users to create rhythmic patterns by toggling steps across multiple drum sample tracks. 


## 📌 Overview
* 16-step sequencer grid
* 7 instrument tracks:
    * Kick
    * Snare
    * Hi-Hat
    * Tom 1
    * Tom 2
    * Clap
    * Ride
* Real-time playback system
* Start / Stop control
* Adjustable BPM (tempo slider)
* Colour-coded tracks for clarity
  
## 🧠 How It Works

Step Sequencer Logic

The sequencer is built using a 2D grid structure:

* Rows represent different instruments
* Columns represent time steps

Each button acts as a toggle:

* ON → sound will play
* OFF → silent
